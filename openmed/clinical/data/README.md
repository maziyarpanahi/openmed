# Bundled clinical data

`zh_cn_administrative_divisions.json` is an OpenMed-maintained compilation of
factual province, city, and district names, distributed under Apache-2.0 with
the rest of this repository. It contains administrative names only: no patient
records, street names, building or room numbers, postcodes, or coordinates.

The intentionally small hierarchy supports shape-correct Chinese surrogates;
it is not a national gazetteer and must not be used for geocoding.
